# Ana Career Guide

Interactive AI-powered career transition guide for worship ministry professionals.

## Tech Stack
- Next.js 16, React 19, TypeScript
- Tailwind CSS 4, Framer Motion
- Vercel AI SDK (Anthropic + Google providers)

## User Preferences

### Workflow
- Autonomous execution, parallel agents when possible
- Haiku/Sonnet only (no Opus)

### Code Standards
- Immutability enforced, 800 line max
- No console.log in committed code
- No hardcoded values
